Colección de dashboards desarrollados con Microsoft Power BI, aplicando modelado de datos, DAX y visualización para convertir datos en decisiones. Cada informe es interactivo - Si quieres explorarlos a placer tienes el enlace a GitHub con los proyectos subidos en el encabezado o en el pie de página.

Dashboards

Dashboard 1 — Panel Ejecutivo

Power BI DAX

Este informe proporciona una visión global de la salud operativa de Adventure Works, centrado en KPIs críticos como Ingreso Total, Rentabilidad y Tasa de Devolución.

Aspectos Técnicos:

  • Navegación Intuitiva: Barra lateral personalizada para una navegación fluida entre las diferentes secciones del informe, mejorando la coherencia visual y la experiencia de usuario.
  • Visualización de Datos Avanzada: Uso de tablas optimizadas con formato condicional y barras de datos integradas en columnas clave, permitiendo una comparación rápida del rendimiento de los 10 productos principales.
  • Análisis Temporal: Visualización de tendencias con indicadores de desviación mes a mes e implementación de métricas DAX complejas para el cálculo preciso de KPIs.

Dashboard 2 — Mapa Caliente

Power BI DAX

Este informe interactivo permite realizar un análisis profundo de la distribución geográfica de las ventas. A través de un mapa de burbujas, se identifica rápidamente el volumen de pedidos por región, facilitando la detección de mercados clave y oportunidades de expansión.

Aspectos Técnicos:

  • Interactividad Avanzada: Implementación de segmentadores (slicers) personalizados que permiten filtrar dinámicamente los datos por territorio (Europa, Norteamérica, Pacífico).
  • Visualización de Datos: Uso de mapas de geolocalización optimizados, donde el tamaño de cada burbuja es proporcional al volumen de ventas.
  • UX/UI: Diseño oscuro (Dark Mode) que resalta los datos clave y reduce la fatiga visual.

Dashboard 3 — Productos

Power BI DAX

Un panel avanzado diseñado para el análisis profundo de productos individuales, permitiendo simular escenarios de precios y medir su impacto directo en la rentabilidad. Este dashboard permite a los gestores de producto tomar decisiones estratégicas basadas en datos históricos y proyecciones.

Aspectos Técnicos:

  • Análisis "What-if": Implementación de parámetros de simulación que permiten ajustar dinámicamente el precio del producto y observar el impacto inmediato en el beneficio total (Adjusted Profit).
  • Modelado de KPIs: Uso de gráficos de tipo medidor (gauge charts) para comparar el rendimiento actual contra objetivos definidos (Target).
  • Inteligencia Narrativa: Panel de resumen inteligente que genera insights automáticos, facilitando la interpretación de métricas clave.

Dashboard 4 — Clientes

Power BI DAX

Este informe integra una arquitectura de navegación avanzada diseñada para maximizar el espacio de visualización sin sacrificar la profundidad del análisis.

Aspectos Técnicos:

  • Arquitectura de Bookmarks: Implementación de un sistema de navegación mediante marcadores (bookmarks) que permite cambiar dinámicamente entre vistas de información, optimizando el espacio del dashboard.
  • Narrativa Condicional: Integración de paneles de información que se activan bajo demanda, mostrando textos descriptivos específicos según la vista seleccionada por el usuario.
  • UX/UI Estratégico: Diseño enfocado en la reducción de ruido visual, permitiendo que el usuario pase de una visión global a una narrativa detallada mediante interacciones intuitivas (botones de acción).

Dashboard 5 — Diccionario

Power BI DAX

La calidad de un modelo de datos se mide por su capacidad de ser mantenido y escalado. Esta página actúa como un catálogo centralizado de medidas DAX, garantizando que el modelo sea transparente y fácil de auditar.

Aspectos Técnicos:

  • Gobernanza de Datos: Documentación exhaustiva que incluye la descripción lógica, la expresión DAX y la ubicación de cada medida, vital para el trabajo colaborativo.
  • Transparencia Analítica: Catálogo interactivo que permite filtrar medidas por tipo o nombre, facilitando la validación rápida de la lógica de negocio aplicada.
  • Buenas Prácticas: Aplicación de estándares de desarrollo (como el uso de tablas dedicadas para medidas) que simplifican el mantenimiento y evitan la deuda técnica en el modelo.

Dashboard 1 — Principal

Power BI DAX

Este informe constituye la solución integral de inteligencia de negocio para MavenMarket, consolidando múltiples fuentes de datos para ofrecer una visión 360° del rendimiento retail. El objetivo es transformar datos transaccionales brutos en decisiones estratégicas sobre inventario, expansión regional y rentabilidad de marca.

Desglose Técnico y Funcional:

  • Monitorización de KPIs en Tiempo Real: Paneles superiores (Cards) que muestran las transacciones, beneficios y devoluciones del mes actual, comparándolos automáticamente contra objetivos (targets) mediante medidas DAX de inteligencia temporal.
  • Análisis de Rendimiento de Marca: Tabla maestra de alta densidad que utiliza formato condicional para evaluar simultáneamente Total Transactions, Profit, Profit Margin y Return Rate, permitiendo identificar rápidamente marcas con baja rentabilidad o altos ratios de devolución.
  • Inteligencia Geográfica: Implementación de mapas interactivos que visualizan la distribución de ventas a nivel global, permitiendo a la gerencia identificar clústeres de mercado y oportunidades de expansión territorial.
  • Análisis de Estructura de Ventas: Uso de Treemaps dinámicos que categorizan las ventas por ubicación, facilitando la detección jerárquica de mercados clave en un entorno visualmente intuitivo.
  • Tendencias de Ingresos: Gráfico de líneas temporales (Weekly Revenue Trending) diseñado para el seguimiento de la estacionalidad, permitiendo prever fluctuaciones en los ingresos y ajustar la cadena de suministro según la demanda histórica.
